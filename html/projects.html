Planted

A mobile app that helps you keep your plants happy. Raspberry Pi collects data from a variety of sensors and sends it to the server. Our React Native app visulizes the data.

Technologies: React Native, React, Redux, D3, Node, GraphQL, PostgreSQL, Javascript, Heroku, Python

Hardware: Raspberry Pi 3 model B, Arduino Uno, OSEPP Humidity and Temperature Sensor - HUMI-01, Generic photo resistor
